# Contributing

This module is currently in early development.

## Development Notes

- Keep the Foundry module id as `vital-quest`.
- The module folder inside any release zip must also be named `vital-quest`.
- Update `version` in `module.json` before making a release.
- Test in a blank Foundry world before uploading a release for wider use.

## Release Checklist

1. Update `module.json` version.
2. Update `CHANGELOG.md`.
3. Build the release zip.
4. Attach both `module.json` and `vital-quest.zip` to the GitHub release.
5. Test installing from the manifest URL.
